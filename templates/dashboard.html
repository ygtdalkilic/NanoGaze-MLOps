LLM Prompt
What is this?
This is the instruction sent to Gemini for every piece of content the agent finds. It decides how credibility is scored.

Required output format
Your prompt must instruct Gemini to respond with exactly these two lines:
credibility: high | medium | low
reason: <one sentence>
Placeholder
Use {text} where you want the article content injected.
Search Queries
What is this?
These are the DuckDuckGo search terms the agent uses to fetch content from the web. Each line is a separate search — results from all queries are combined and sent through the hallucination filter.

Format rules
• One query per line
• Blank lines are ignored
• Be specific — vague queries return noisier results

How the pipeline uses these
1 · Agent
Runs your search queries against DuckDuckGo and collects articles.
2 · LLM Analysis
Each article is passed through your prompt to get a credibility score from Gemini.
3 · Engine 1
Calculates a trust score using signals (IPs, statistics, claims, source).
4 · Engine 2
Cross-validates LLM output against Engine 1. Mismatches are flagged as threats.
